#Приложение `ask me`

Приложение предназначено для обмена сообщениями в формате вопрос-ответ на публичной странице профиля пользователя.

**Ruby v:** `>= 2.4.0`

**Rails v:** `~> 4.2.9`

**Gemset list:**

gem 'rails-i18n'

gem 'rails_12factor'

gem 'uglifier'

gem 'jquery-rails'

gem 'pg' - only production

gem 'byebug'

gem 'sqlite3'

gem 'web-console', '~> 2.0'
